How to use this admin tool

ADDING A NEW PRODUCT

  1. Click Add / Edit Product tab
  2. Fill in name, collection, price, badge
  3. Write the description and bullet points
  4. Upload your product photo to GitHub first: images/your-product.png
  5. Enter the image filename
  6. Add variants (Standard, Personalised etc.)
  7. Add Stripe Payment Links for each variant
  8. Tick "Show at the Stall" if relevant
  9. Click Save Product
  10. Go to Export Code tab
  11. Click Generate products.js
  12. Download and push to GitHub

EDITING A PRODUCT

  1. Go to Products tab
  2. Click Edit on the product you want to change
  3. Make your changes
  4. Click Save Product
  5. Export and push updated products.js

UPLOADING PHOTOS TO GITHUB

  1. Go to github.com/GHaigh/slatethelakes
  2. Click into the images folder
  3. Click Add file → Upload files
  4. Drag your product photo in
  5. Commit — use filename like my-product.png
  6. Enter that filename in the admin form

💾 Your data is saved in this browser

Products you add are saved in your browser's localStorage, so they persist between sessions on the same computer. Always export and push to GitHub to make changes live on the site.
